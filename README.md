
Scatterplot Graph

A simple data visualization project that generates Scatter Plot Graphs using Python. This project helps visualize relationships between two variables through plotted data points.

📌 Features
Generate scatter plot graphs
Visualize relationships between datasets
Customizable graph appearance
Axis labels and titles
Beginner-friendly implementation

Python
Matplotlib
NumPy
Pandas

Run the project using:

python main.py
📊 Example Scatter Plot

A scatter plot represents data points on a graph to show relationships between variables.

Example:

import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 4, 5, 4, 5]

plt.scatter(x, y)
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.title("Scatter Plot Graph")
plt.show()
